lotomidas.dexterity Installation
--------------------------------

To install lotomidas.dexterity using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``lotomidas.dexterity`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        lotomidas.dexterity
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
